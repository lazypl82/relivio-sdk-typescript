# Relivio TypeScript SDK

Primitive TypeScript SDK for Relivio v0.1.

Current scope:
- register deployment
- send single ingest log
- send ingest batch
- explicit exception capture helper
- protection status read for guard code
- read latest verdict for service-side guard flows
- local SDK status

Non-goals:
- framework middleware
- framework adapters
- request object parsing
- auto instrumentation
- guard logic
- full verdict consumer surface
